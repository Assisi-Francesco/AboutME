# 👄 입코딩 (Ip-Coding)

> **"손가락은 거들 뿐, 코딩은 입으로 한다."**
> 
> *The True Developer communicates via voice, not syntax.*

## 📝 프로젝트 소개 (Introduction)

**입코딩(Ip-Coding)**은 손가락 통증에 시달리는 현대 개발자들을 위한 혁신적인(?) 포트폴리오 시뮬레이터입니다.  
이 프로젝트는 "코드는 논리가 아니라 말빨로 짜는 것"이라는 잘못된 신념을 바탕으로 제작되었습니다.

사용자가 마이크 버튼을 누르고 말을 하면, 브라우저가 이를 텍스트로 받아적어 줍니다. (물론 코드로 변환해주지는 않습니다. 그건 AI가 할 일이니까요.)

## ✨ 주요 기능 (Features)

* **🗣️ 실시간 입털기 시뮬레이션 (STT):**
    * JavaScript `Web Speech API`를 활용하여 사용자의 음성을 실시간 텍스트로 변환합니다.
    * 마이크 권한만 있다면 누구나 '입코딩'을 체험할 수 있습니다.
* **📄 솔직한 포트폴리오:**
    * 화려한 경력 대신 '회의 소집 능력', '요구사항 필터링' 등 개발자의 리얼한 스킬셋을 명시했습니다.
* **🎨 반응형 디자인:**
    * CSS Variable을 활용한 블루 톤의 깔끔한 UI/UX.
    * 마이크 활성화 시 심장이 뛰는 듯한 `Pulse` 애니메이션 효과.
